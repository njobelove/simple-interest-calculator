## Overview

The Simple Interest Calculator is a responsive web tool that helps users quickly compute:
- Simple Interest earned on an investment or loan
- Total amount (Principal + Interest)

This calculator is ideal for students learning financial mathematics, professionals making quick calculations, or anyone needing to compute interest without complex software.

## Features

- ✅ **Real-time Calculation**: Instant results with accurate formula implementation
- ✅ **Input Validation**: Ensures all inputs are valid numbers (positive values)
- ✅ **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- ✅ **Error Handling**: User-friendly error messages for invalid inputs
- ✅ **Clear/Reset Function**: One-click reset to clear all fields
- ✅ **Keyboard Support**: Press 'Enter' key to calculate
- ✅ **Smooth Animations**: Modern UI with fade and slide effects
- ✅ **No External Dependencies**: Pure HTML/CSS/JavaScript, no frameworks required
